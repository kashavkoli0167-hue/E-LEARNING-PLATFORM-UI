**PROJECT OVERVIEW** --
The E-Learning Platform UI Project is a frontend-focused web application designed to represent a modern, clean, and user-friendly interface for an online learning system. This project is built using HTML5 and CSS3 only, with the main goal of strengthening core frontend concepts such as layout structuring, reusable components, responsive grids, and modern UI design principles.
The project simulates a real-world e-learning platform where users can explore available courses, view course details, track their learning progress, and watch embedded video lessons — all within a consistent and visually appealing interface. This project is ideal for beginners who want to practice professional UI development without using JavaScript or backend technologies.

**PROJECT OBJECTIVES** --
The primary objectives of this project are:
- To practice semantic HTML structure
- To build a multi-page website with consistent navigation
- To understand CSS layouts using Flexbox and Grid
- To implement modern UI effects like gradients, hover animations, and cards
- To design a clean and professional interface suitable for an internship-level project

  This project focuses more on user interface and experience (UI/UX) rather than functionality, making it a strong foundation for future JavaScript or backend integration.

**FEATURES** --
The E-Learning Platform UI includes the following key sections:
1. Courses Page (Home Page) --
The landing page displays a list of available courses using interactive course cards. Each card represents a course category and navigates the user to a dedicated course detail page. Hover effects and smooth transitions enhance the overall user experience.
2. Course Detail Pages --
Each course detail page provides an overview of the course content, including topics covered and learning outcomes. The layout is designed using reusable card components to maintain visual consistency across the platform.
3. Progress Tracking Page --
The progress page visually represents the learner’s progress using horizontal progress bars. Each progress bar indicates the completion percentage of a specific course, helping users easily track their learning status.
4. Video Lessons Page --
This section embeds YouTube videos directly into the platform using iframes. Videos are arranged in a responsive grid layout, allowing users to watch learning content without leaving the website.
5. Navigation Bar & Footer --
A fixed navigation bar provides seamless navigation between pages. The footer includes project branding and developer credit, adding a professional touch to the application.

**TECHNOLOGY USED**
- HTML5 – for semantic structure and content organization
- CSS3 – for styling, layouts, gradients, animations, and responsiveness

  No external libraries or frameworks were used, ensuring a strong focus on core frontend skills.

**DESIGN APPROCH** --
The UI design follows a dark theme with multi-color gradient accents, giving the project a modern and polished appearance. Glassmorphism-style cards, subtle shadows, and hover animations were used to improve visual hierarchy and interactivity.
- Special attention was given to:
- Clean spacing and alignment
- Reusable CSS classes
- Consistent color palette
- Beginner-friendly but professional design

**LEARNING OUTCOMES** --
By building this project, the following skills were practiced and improved:
- Structuring multi-page websites
- Writing clean and maintainable CSS
- Creating reusable UI components
- Understanding layout systems (Grid & Flexbox)
- Improving UI/UX thinking for real-world applications
